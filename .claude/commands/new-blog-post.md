# Create New Blog Post

Create a new blog post following the style guide at `.claude/blog-writing-style.md`.

## Instructions

1. Read `.claude/blog-writing-style.md` first
2. Create the file at `content/blog/YYYY-MM-DD-slug.md` using today's date
3. Include proper front matter with title, description, tags, and subtitle
4. Add `<!-- more -->` marker after the introduction
5. Follow the heading structure (limit h2 to 5-7, never skip levels)
6. End with a `## Related` section
