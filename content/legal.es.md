# Aviso Legal y Privacidad

<p class="legal-updated">Última actualización: Diciembre 2025</p>

## Titular del Sitio

Este sitio web está operado por **Jose Maria Valera Reales** (Chemaclass).

Para contacto, puedes encontrarme en [GitHub](https://github.com/Chemaclass).

## Analíticas

Este sitio utiliza [Cronitor RUM](https://cronitor.io/real-user-monitoring) para recopilar estadísticas de uso anónimas. Esto me ayuda a entender cómo los visitantes usan el sitio y mejorar el contenido.

**Los datos recopilados incluyen:**
- Páginas visitadas
- Ubicación geográfica aproximada (país/región)
- Tipo de dispositivo y navegador
- Información de referencia

Estos datos son agregados y no te identifican personalmente.

## Contenido Incrustado

Algunas páginas pueden incluir:

- **Vídeos de YouTube** — Cuando ves una página con un vídeo incrustado, YouTube puede establecer cookies y recopilar datos según la [Política de Privacidad de Google](https://policies.google.com/privacy).
- **GitHub Gists** — Los fragmentos de código incrustados desde GitHub pueden cargar scripts externos según la [Política de Privacidad de GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## Servicios de Terceros

Este sitio utiliza los siguientes servicios externos:

- **FontAwesome** — para iconos
- **PayPal/Alby** — para donaciones (enlaces externos)
- **GitHub** — para alojamiento de código y enlaces

Cada servicio tiene su propia política de privacidad.

## Almacenamiento Local

Este sitio guarda tu preferencia de tema (modo claro/oscuro) en el almacenamiento local de tu navegador. Esto no es una cookie y no se transmite a ningún servidor.

## Tus Derechos

Bajo el RGPD, tienes derecho a:
- Saber qué datos se recopilan sobre ti
- Solicitar la eliminación de tus datos
- Optar por no ser rastreado

Dado que Cronitor RUM recopila datos anonimizados, no hay datos personales que eliminar. Puedes bloquear las analíticas usando una extensión del navegador como uBlock Origin o desactivando JavaScript.

## Descargo de Responsabilidad

Los artículos y publicaciones de este sitio reflejan mis opiniones y experiencias personales. Aunque ofrezco [servicios profesionales](/es/services/) de desarrollo web y talleres para equipos, el contenido gratuito publicado aquí es de carácter educativo y no sustituye una consultoría formal.

No soy responsable del contenido de los sitios web externos enlazados desde este sitio.
