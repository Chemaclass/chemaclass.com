Talleres de Equipo

Sesiones prácticas de código que potencian a tu equipo

Pasa un día completo programando, testeando y refactorizando. Sesión de Mañana

Testing & TDD

Nos enfocamos en técnicas de testing que hacen el cambio seguro, desde tests golden master que capturan el comportamiento existente hasta ejercicios TDD que guían mejor diseño.

Temas cubiertos

  • Testing golden master
  • Patrones de approval testing
  • Ciclo de desarrollo guiado por tests
  • Escribir tests que guían el diseño
  • Entendiendo los test doubles

Sesión de Tarde

Refactoring

Cambiamos el enfoque al refactoring: aplicando cambios pequeños y focalizados para mejorar código legacy sin romperlo.

Temas cubiertos

  • Técnicas de refactoring seguro
  • Patrones extract method & class
  • Trabajar con código legacy
  • Identificación de code smells
  • Estrategias de mejora incremental

Qué ganará tu equipo

Confianza en testing

  • Escribir tests significativos
  • Testear código legacy con seguridad
  • Crear el hábito de testear

Mentalidad TDD

  • Ciclo red-green-refactor
  • Diseñar a través de tests
  • Commits más pequeños y focalizados

Habilidades de refactoring

  • Mejorar código con seguridad
  • Abordar deuda técnica
  • Patrones de código limpio

Resultado clave: Tu equipo sale con un toolkit probado para escribir mejor código, mejorar codebases existentes y colaborar efectivamente mediante pair programming.
